Node.js Online Minigames

Using Node.js along with Socket.io I am creating an online game that you can play with friends. To display information to the client I am using a collection of pure HTML and Javascript canvas elements. You will be able to create lobbies and control exactly how the game is played.

It will start having three different minigames, where you try to collect as much money as you can before the end of the game. If enough people enjoy the concept I will expand the number of mini-games.
