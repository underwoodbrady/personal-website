Commission Website

I made a commission website for a friend using Dreamhost and Wordpress to allow him to make changes after the site is published. The fun part about this project was having full creative freedom to not only design the layout but also the logo and icon for the site. I wrote several mock-up designs in Brackets and made the logo in illustrator.

This project was a good learning experience as I had never worked with a CMS before and it was enlightening to see the process of creating a website from start to finish.
