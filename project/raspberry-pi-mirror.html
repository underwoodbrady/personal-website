Raspberry Pi Smart Mirror

I created this smart mirror for a DECA competition through my high school. It was made with two-way glass, a monitor, and a Raspberry Pi 3 Model B+, along with core software developed by MagicMirror, an open source modular smart mirror platform.

I altered the code to fit my design and was able to create a working smart mirror that automatically starts up when plugged in and can display all your important information in one place.

For more information on MagicMirror visit https://magicmirror.builders
